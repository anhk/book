
# 密码标准框架

| 标准级别         | 标准标识 | 描述                                                         |
| ---------------- | -------- | ------------------------------------------------------------ |
| 密码行业标准     | GM/T     | 密码行业标准化技术委员会（密标委）主导，密标委负责密码技术、产品、系统、管理等方面的标准化工作。[传送门](http://www.gmbz.org.cn/main/bzlb.html) |
| 密码行业国家标准 | GB/T     | 由全国信息安全标准化技术委员会（信安标委）WG3工作组为依托，将密码行业标准陆续转化为国家标准。[传送门](https://www.tc260.org.cn/) |
| 国际标准         | -        | -                                                            |
